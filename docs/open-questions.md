# Открытые вопросы и решения

Живой список. Решённое перемещаем в раздел «Принятые решения» с датой.

## Открытые

- **Модель эмбеддингов.** Локальная multilingual-модель (например, multilingual-e5 / BGE-m3,
  on-premise, данные не уходят наружу) против внешнего API эмбеддингов (проще, но документы
  уходят провайдеру). Документы и вопросы - на русском. Решить на этапе прототипа векторизации.
- **Форматы исходных инструкций.** Подтверждены Word и PDF; полный перечень не собран.
  Нужен ли OCR или мультимодальная обработка для инструкций со скриншотами интерфейса 1С -
  уточнить, когда соберём образцы документов.
- **Порог сходства для «белого списка».** Значение и метрика подбираются на прототипе.
- **Хостинг LLM.** OpenRouter (облачные модели) - зафиксировано для пилота. Соответствие
  политике безопасности (какие данные уходят в облако) - проверить перед продом.
- **Аутентификация и разграничение доступа** на портале и в админском модуле - не проработано.
- **Разделение баз знаний по подразделениям** (при подключении других служб) - продумать модель.
