  - qwen многословнее и потому дороже на выходе ($1.10/M против $0.35/M), но речь о

  - запрет markdown - gemma возвращала `**жирный**`, а портал показывает текст как есть
    (звёздочки были бы видны пользователю);
  - запрет односложных ответов - qwen на «можно ли поменять НД в договоре?» отвечал

    модели дают полный ответ. Прогон повторён на исправленном промпте - сравнение честное.
- Грабли: незавершающиеся python-процессы снова помешали - два прогона в одной

  создал файл вывода). Запускать прогоны по одному, при странностях - проверять
