## Решения, которые нужно принять до кода

Не «додумываем» молча - выносим владельцу.

1. **Значения enum коллекций.** Предлагается `erp` и `zup` (короткие латинские
   коды в БД) с отображаемыми названиями «1С:ERP» и «1С:ЗУП» в UI.
2. **Куда отнести старый корпус `docs/manuals/IT_1C`** (8 инструкций, на них
   калибровался порог и построен eval). Предложение: это инструкции по ERP -
   отнести к коллекции `erp`. Альтернатива - отдельная коллекция «внутренние
   инструкции УПЗ», но тогда пользователь должен выбирать её в UI отдельно.
3. **Раскладка папок инструкций.** Сейчас ИТС-ERP выгружен в `docs/manuals/parsed`.
   Предложение: `docs/manuals/its_erp/` и `docs/manuals/its_zup/` (коллекция =
   папка, `--collection` задаётся явно). Переименование папки меняет `source_path`,
   а ingest ищет документ по `source_path` (`pipeline.py:53-55`) - старые строки
   останутся сиротами, их надо будет удалить и переиндексировать (см. B3).
4. **Порог `SIMILARITY_THRESHOLD` - общий или свой на коллекцию.** 0.53
   откалиброван на корпусе ERP. Для ЗУП нужна проверка; если оптимум разойдётся -
   порог станет свойством коллекции.

## Риски

- **R1. Фильтрация + HNSW.** Векторный индекс ищет ближайших по всей таблице, а
  `WHERE collection=...` отсекает результаты **после** индекса: при сильном перекосе
  размеров коллекций top-k может «съесться» соседней базой и вернуть меньше k
  релевантных. В pgvector 0.8 для этого есть итеративное сканирование
  (`hnsw.iterative_scan`), альтернатива - частичные индексы по коллекциям.
  Проверить на реальных данных (E4), решение зафиксировать в open-questions.
- **R2. Объём ЗУП неизвестен.** Оценка появится после A1 (по числу строк реестра).
  Для ERP: 3645 строк -> 411 PDF -> 800 МБ -> около 10 часов парсинга и 1,5 часа
  индексации.
- **R3. Расползание `source_path`.** Любое переименование папок инструкций
  порождает сиротские строки в БД (ingest сверяет по пути). Чистить осознанно
  (B3), иначе поиск будет ссылаться на несуществующие файлы.
- **R4. Порог, откалиброванный на ERP, может не подойти ЗУП** (E3).
